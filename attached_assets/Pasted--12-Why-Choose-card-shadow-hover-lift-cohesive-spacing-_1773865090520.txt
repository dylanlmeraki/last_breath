#12 - Why Choose card: shadow, hover lift, cohesive spacing
What & Why
Wrap the left-column content of the "Why Pacific Engineering?" section in a card container with shadow, hover-lift interaction, and tightened internal spacing for visual cohesion. Currently the content (title, description, feature list, buttons) floats openly in the grid column — giving it a card shell will add structure and visual weight consistent with the engineering aesthetic.

Done looks like
Left column content is visually contained in a rounded card with a visible shadow behind it
On hover, the card lifts slightly (translateY ~4px) with the shadow deepening — smooth 300ms transition
Internal spacing between title → accent line → description → feature items → CTA buttons is balanced and cohesive (no orphaned gaps or cramped areas)
Card has consistent padding on all sides
No visual regressions on the right column (image + badge), transitions, or other sections
Existing scroll animations (AnimatedSection, motion.div) still work correctly inside the card
Out of scope
Text copy changes
Mobile layout restructuring
Changes to the right column (image/badge)
Changes to other sections (Hero, Services, CTA, Footer)
Nav or hero modifications
